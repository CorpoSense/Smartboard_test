SmartBorad

Everything about us :

      Description:

        SmartBoard is a smart business dashboard that allows users to apply one or more Data Science algorithms to a dataset (e.g. Clustering) to solve a business         data problem (such as customer segmentation ... etc).

      Objectif:

        Provide an easy to use yet powerful solution for Data Analytics using a predefined algorithms and ready to use data model for most common data problem.

      How it works?

  • Companies can upload their data to the dashboard (via http or ftp) in different formats (csv, xlsx)
  • They can then select an appropriate algorithm (Clustering) to solve a specific business data problem (things like: Customer Segmentation)
  • Generate a Machine Learning model for different purposes ( Classification…etc.)
  • Save the generated data model for further analysis (each user/company has his own space to store and restore data model)
  • Use the predefined model on a new uploaded dataset for analysis.

      Targeted audience:

  • Companies from different industries including: pharmacy, telecom, food industry.Other companies can use solution if it fit their needs.
  • Business model: this solution will be provided for free as SAAS, however; the uploaded dataset is limited, companies can subscribe for dedicated hostingor self-hosting, consultancy can be provided if needed, the full code source is provided in public hosting service.

      Technical details:

  • The full solution will be written in Python 3, using the most common open source libraries.
  • We used python Web framework Flask with many other extensions.
  • The front-end tier is provided (template, css…), however the user experience should be simplified for non-technical users with more visualization libraries, charts & graphs should be included to provide users with more illustration so they can judge the quality of the output easily
  • Heavy tasks (things like Deep learning, neural nets…) should be scheduled using a pool of worker process which will notify the user once it’s finished (Pool worker is allowed only for premium user).
